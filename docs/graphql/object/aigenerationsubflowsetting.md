Represents a sub-flow setting generated by AI.

## Fields without arguments

| Name | Type | Description |
|------|------|-------------|
| `defaultValue` | [`JSON`](../scalar/json.md) | The generated default value of the sub-flow setting. |
| `hidden` | [`Boolean`](../scalar/boolean.md) | Whether the generated sub-flow setting is hidden. |
| `identifier` | [`String`](../scalar/string.md) | The generated sub-flow setting identifier. |
| `optional` | [`Boolean`](../scalar/boolean.md) | Whether the generated sub-flow setting is optional. |
