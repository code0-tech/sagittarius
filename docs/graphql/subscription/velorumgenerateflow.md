## Arguments

| Name | Type | Description |
|------|------|-------------|
| `id` | [`String!`](../scalar/string.md) | Velorum generation request identifier returned by the mutation |

## Fields

| Name | Type | Description |
|------|------|-------------|
| `flow` | [`JSON`](../scalar/json.md) | Generated flow returned by Velorum |
